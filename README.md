# AnalisisNumeros

## Descripción

`AnalisisNumeros` es un programa en Java que analiza un conjunto de números almacenados en un array.  
Permite realizar tres operaciones principales:

1. Encontrar el número máximo del array.
2. Contar cuántas veces se repite el número máximo.
3. Calcular el promedio de todos los números del array.


## Cómo ejecutar el programa

Compila y ejecuta el archivo `AnalisisNumeros.java` usando tu IDE o la terminal con Java.
